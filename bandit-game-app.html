무임승차 상호작용 시뮬레이터

플레이어와 단계를 정하고, 협업 도식과 무임승차 전략을 설정한 뒤 결과를 확인하세요.

1. 기본 설정

각 단계에서 한 노드에 배정된 플레이어들이 하나의 협업 그룹이 됩니다.

2. 협업 도식과 플레이어 배정

간선:같은 행의 점선을 클릭해 굵은 실선으로 만드세요. 굵은 선으로 이어진 노드들이 하나의 분업 그룹이 됩니다.
대응 설정

1단계의 플레이어 1부터 순서대로, 대응할 노드를 클릭하세요. 한 행의 모든 플레이어를 대응시키면 다음 행으로 자동 이동합니다. 각 행에서 굵은 선으로 연결된 노드의 묶음이 그 단계의 분업 단위로 시뮬레이션에 반영됩니다.

3. 그룹별 상호작용 선택

한 그룹에서 여러 상호작용을 함께 선택할 수 있습니다. 굵은 선을 추가하거나 해제하면 아래 그룹도 자동으로 갱신됩니다.

4. 무임승차자와 전략

무임승차자는 선택하지 않아도 됩니다. 선택한 플레이어만 전략을 정합니다.

0협업 그룹
